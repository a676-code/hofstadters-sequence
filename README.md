# hofstaders-sequence
This script generates [Hofstadter's sequence](https://www.youtube.com/watch?v=j0o-pMIR8uk). 

output: `[1, 1, 2, 3, 3, 4, 5, 5, 6, 6, 6, 8, 8, 8, 10, 9, 10, 11, 11, 12]`
